# 🌙 Luna Fortuna Bot

ربات تلگرام فروشگاه Luna Fortuna

## امکانات مشتری
- ثبت سفارش با لینک
- محاسبه قیمت (لیر به تومان + ۱۵٪ کارمزد)
- درخواست پیدا کردن محصول
- پیگیری وضعیت سفارش
- مشاهده حراج‌های امروز

## امکانات ادمین
- مشاهده سفارش‌های جدید
- تأیید/رد پرداخت
- آپدیت وضعیت سفارش
- تنظیم نرخ لیر
- آمار روزانه
- پیام همگانی

## تنظیمات Environment Variables در Vercel
- `BOT_TOKEN` — توکن ربات از BotFather
- `ADMIN_CHAT_ID` — آیدی عددی تلگرام ادمین

## نحوه پیدا کردن ADMIN_CHAT_ID
به ربات @userinfobot در تلگرام پیام بده — آیدی عددیت رو میده
